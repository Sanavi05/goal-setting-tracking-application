# Hackathon Demo Script

1. Log in as Employee using `employee@goalgrid.demo`.
2. Open Goal Sheet, edit three goals, keep total weightage at exactly 100%, save draft, and submit to manager.
3. Log out and log in as Manager using `manager@goalgrid.demo`.
4. Open Approvals, review submitted sheets, approve one sheet, and point out the lock and audit behavior.
5. Log back in as Employee, open Check-ins, enter Q1 actual achievement and a comment.
6. Log in as Admin using `admin@goalgrid.demo`.
7. Open Dashboard to show completion overview, audit count, and escalation count.
8. Open Cycle Management to show annual cycle/check-in window configuration.
9. Open Reports and export achievement data as CSV or Excel.
10. Open Analytics to show QoQ trends and manager effectiveness.
11. Open Shared Goals to create a departmental KPI and assign it to employees.

Positioning line: GoalGrid is not just a form app; it is an audit-ready performance workflow system.
